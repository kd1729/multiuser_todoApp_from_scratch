# A simple TODO app made from Scratch using ReactJS.
### Key Features

- Add / Delete a Todo.
- Add Deadline if needed.
- Mark Todo as Done or not done.
- Filter todos which are completed.
- Filter Todos which are pending.
- Delete the marked todos all at once.
